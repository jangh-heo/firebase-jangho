# Project Blueprint - Greeting App

## Overview
A modern, framework-less web application designed to display a personalized greeting with high-quality aesthetics and modern web standards.

## Project Details
- **Architecture:** Vanilla HTML, CSS, and JavaScript.
- **Components:** Custom Web Components for encapsulated UI.
- **Styling:** Modern CSS (Baseline) featuring container queries, `:has()` selector, and vibrant color palettes with **Dark Mode support**.
- **Interactivity:** Smooth transitions, responsive design, and theme persistence.
