# Project Blueprint - Greeting App

## Overview
A modern, framework-less web application designed to display a personalized greeting with high-quality aesthetics and modern web standards.

## Project Details
- **Architecture:** Vanilla HTML, CSS, and JavaScript.
- **Components:** Custom Web Components for encapsulated UI.
- **Styling:** Modern CSS (Baseline) featuring container queries, `:has()` selector, and vibrant color palettes.
- **Interactivity:** Smooth transitions and responsive design.

## Current Task: Personalized Greeting
Implement a "Hello Jangho" (안녕 장호야) greeting with a premium look and feel.

### Steps:
1.  **Update `index.html`**: Set up a clean entry point and include necessary metadata.
2.  **Update `style.css`**: Implement a vibrant, modern background with noise texture, expressive typography, and centered layout.
3.  **Update `main.js`**: Create a custom Web Component `<jangho-greeting>` to display the text.
4.  **Verification**: Ensure the preview renders correctly without errors.
